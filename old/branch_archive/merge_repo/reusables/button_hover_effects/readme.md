# Button Hover Effects



Reusable CSS button hover effects for your frontend projects.

## Effects Included
- **Glow** (`btn-glow`): Adds a glowing shadow on hover.
- **Slide** (`btn-slide`): Animated background slides in on hover.
- **Scale** (`btn-scale`): Button scales up on hover.
- **Border** (`btn-border`): Border and text color change on hover.
- **Shadow** (`btn-shadow`): Box shadow intensifies on hover.
- **Rotate** (`btn-rotate`): Button rotates and scales on hover.
- **Ripple** (`btn-ripple`): Ripple animation on hover.
- **Gradient** (`btn-gradient`): Background gradient shifts on hover.
- **Flip** (`btn-flip`): Button flips to reveal alternate text.
- **Pulse** (`btn-pulse`): Button gently pulses on hover.
- **Outline Fill** (`btn-outlinefill`): Outline fills with color on hover.

## How to Use



1. **Import the CSS file for the effect you want:**
	```html
	<link rel="stylesheet" href="glow.css">
	<link rel="stylesheet" href="slide.css">
	<link rel="stylesheet" href="scale.css">
	<link rel="stylesheet" href="border.css">
	<link rel="stylesheet" href="shadow.css">
	<link rel="stylesheet" href="rotate.css">
	<link rel="stylesheet" href="ripple.css">
	<link rel="stylesheet" href="gradient.css">
	<link rel="stylesheet" href="flip.css">
	<link rel="stylesheet" href="pulse.css">
	<link rel="stylesheet" href="outlinefill.css">
	```
	(Adjust the path as needed)

2. **Add the corresponding class to your button:**
	```html
	<button class="btn-glow">Glow Effect</button>
	<button class="btn-slide"><span>Slide Effect</span></button>
	<button class="btn-scale">Scale Effect</button>
	<button class="btn-border">Border Effect</button>
	<button class="btn-shadow">Shadow Effect</button>
	<button class="btn-rotate">Rotate Effect</button>
	<button class="btn-ripple"><span>Ripple Effect</span></button>
	<button class="btn-gradient">Gradient Effect</button>
	<button class="btn-flip"><span class="btn-flip-inner"><span class="btn-flip-front">Flip Front</span><span class="btn-flip-back">Flip Back</span></span></button>
	<button class="btn-pulse">Pulse Effect</button>
	<button class="btn-outlinefill"><span>Outline Fill Effect</span></button>
	```
