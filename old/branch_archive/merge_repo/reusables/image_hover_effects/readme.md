## Image Hover Effects


Reusable CSS image hover effects for your frontend projects.

### Effects Included
- **Zoom-in** (`img-zoom`): Image scales up on hover.
- **Grayscale to Color** (`img-grayscale`): Image transitions from grayscale to color.
- **Shadow Glow** (`img-shadow`): Shadow appears on hover.
- **Border Animation** (`img-border`): Border and subtle glow appear on hover.
- **Rotate** (`img-rotate`): Image rotates and scales on hover.
- **Blur** (`img-blur`): Image blurs and brightens on hover.
- **Scale Up Down** (`img-scaleupdown`): Image scales and skews on hover.
- **Fade** (`img-fade`): Image fades (opacity changes) on hover.


### How to Use
1. **Import the CSS file for the effect you want:**
   ```html
   <link rel="stylesheet" href="zoom.css">
   <link rel="stylesheet" href="grayscale.css">
   <link rel="stylesheet" href="shadow.css">
   <link rel="stylesheet" href="border.css">
   <link rel="stylesheet" href="rotate.css">
   <link rel="stylesheet" href="blur.css">
   <link rel="stylesheet" href="scaleupdown.css">
   <link rel="stylesheet" href="fade.css">
   ```
   (Adjust the path as needed)

2. **Add the corresponding class to your image:**
   ```html
   <img src="your-image.jpg" class="img-zoom" alt="Zoom Effect">
   <img src="your-image.jpg" class="img-grayscale" alt="Grayscale Effect">
   <img src="your-image.jpg" class="img-shadow" alt="Shadow Effect">
   <img src="your-image.jpg" class="img-border" alt="Border Effect">
   <img src="your-image.jpg" class="img-rotate" alt="Rotate Effect">
   <img src="your-image.jpg" class="img-blur" alt="Blur Effect">
   <img src="your-image.jpg" class="img-scaleupdown" alt="Scale Up Down Effect">
   <img src="your-image.jpg" class="img-fade" alt="Fade Effect">
   ```
