
# Link Hover Effects

Reusable CSS link hover effects for your frontend projects.

## Effects Included
- **Underline Animation** (`link-underline`): Underline smoothly animates in on hover.
- **Color Fade** (`link-colorfade`): Link text color transitions smoothly on hover.
- **Background Highlight** (`link-bgslide`): A background highlight slides in from the left or right on hover.

## How to Use
1. **Import the CSS file for the effect you want:**
   ```html
   <link rel="stylesheet" href="link-underline.css">
   <link rel="stylesheet" href="link-colorfade.css">
   <link rel="stylesheet" href="link-bgslide.css">
   ```
   (Adjust the path as needed)

2. **Add the corresponding class to your link:**
   ```html
   <a href="#" class="link-underline">Underline Animation</a>
   <a href="#" class="link-colorfade">Color Fade</a>
   <a href="#" class="link-bgslide">Background Highlight</a>
   ```
